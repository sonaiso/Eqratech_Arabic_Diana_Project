# Masdar Semantic Analysis - Phonetic-Semantic Mapping System

## نظام التحليل الدلالي للمصادر - الربط الصوتي-الدلالي

This document provides a comprehensive framework for analyzing Arabic verbal nouns (مصادر) through a layered semantic classification system that connects morphological patterns with semantic domains.

---

## 1. Layered Semantic Classification (التصنيف الدلالي الطبقي)

### Semantic Domains (المجالات الدلالية)

Arabic verbal nouns are classified into the following semantic domains:

#### Primary Domains:

1. **MasdarPhysical (حسي)** - Physical/Sensory domain
   - Concrete, tangible actions or states
   - Examples: ضرب (hitting), كسر (breaking), قطع (cutting)

2. **MasdarDynamic (فيزيائي - حركي)** - Dynamic/Movement domain
   - Motion, transition, physical effects
   - Examples: صعود (ascending), نزول (descending), ركوب (riding)

3. **MasdarCognitive (عقلي)** - Cognitive/Mental domain
   - Understanding, knowledge, perception
   - Examples: علم (knowledge), فهم (understanding), حفظ (memorization)

4. **MasdarEmotional (شعوري)** - Emotional/Affective domain
   - Fear, anger, sadness, joy
   - Examples: فرح (joy), غضب (anger), حزن (sadness)

5. **MasdarSocial (اجتماعي)** - Social/Interactional domain
   - Fighting, quarreling, social dealings
   - Examples: قتال (combat), خصام (dispute), تعامل (dealing)

6. **MasdarTask (وظيفي)** - Task/Process domain
   - Writing, farming, manufacturing
   - Examples: كتابة (writing), زراعة (agriculture), صناعة (manufacturing)

7. **MasdarExistentialState (وجودي-حالي)** - Existential/State domain
   - Standing, sitting, sleeping
   - Examples: قيام (standing), جلوس (sitting), نوم (sleeping)

#### Traditional Categories (retained):

- **MasdarEvent** - General event
- **MasdarStateMovement** - State/movement
- **MasdarProcess** - Process
- **MasdarEffect** - Effect/impact
- **MasdarInteraction** - Interaction
- **MasdarOnce** - Single instance (اسم المرة)
- **MasdarType** - Type/manner (اسم الهيئة)
- **MasdarProfession** - Profession/craft

## 2. Phonetic-Semantic Correlation (الربط الصوتي-الدلالي)

### The Golden Rule: Vowel Patterns Carry Semantic Weight

Arabic verb patterns (أبواب الفعل) have inherent semantic tendencies based on vowel patterns:

| Verb Pattern | Eye Vowel | Semantic Tendency | Phonetic Interpretation |
|-------------|-----------|-------------------|------------------------|
| **فَعَلَ** | Fatha (a) | General events, open/unrestricted actions | Openness (الانفتاح) = generality |
| **فَعِلَ** | Kasra (i) | Cognitive/emotional states | Compression (الانضغاط) = internalization → mental/emotional |
| **فَعُلَ** | Damma (u) | Physical/bodily attributes or stable states | Fullness (الامتلاء) = stability/permanence |

### Key Phonetic-Semantic Principles:

1. **Kasra (الكسرة) in verb middle** → Cognitive/emotional meaning
2. **Damma (الضمة) in verb middle** → Stable attributes/states
3. **Fatha (الفتحة) in verb middle** → Physical events or general actions

## 4. Detailed Pattern Semantics (الدلالات التفصيلية للأوزان)

### 1. Pattern: **فَعْل** (fa'l)

**Semantic Profile:**
- **Domains:** Physical, Impact
- **Characteristics:** Sharp, concrete, material effect
- **Vowel Structure:** Fatha + Sukun
- **Semantic Features:**
  - Agent force: High (0.9)
  - Patient effect: Very High (1.0)
  - Physicality: Very High (1.0)
  - Cognition: Low (0.1)

**Examples:**
- قَتْل (killing) - direct physical action
- كَسْر (breaking) - material destruction
- قَطْع (cutting) - physical separation
- ضَرْب (hitting) - impact action

---

### 2. Pattern: **فُعُول** (fu'ūl)

**Semantic Profile:**
- **Domains:** Dynamic, Movement, Physical State
- **Characteristics:** Extended, continuous, habitual
- **Vowel Structure:** Damma + Damma
- **Semantic Features:**
  - Movement: High (0.9)
  - Extension: Very High (1.0)
  - Physicality: High (0.8)
  - Habituality: High (0.9)

**Examples:**
- صُعُود (ascending) - upward movement
- نُزُول (descending) - downward movement
- جُلُوس (sitting) - postural state
- رُكُوب (riding) - movement on vehicle

---

### 3. Pattern: **فِعَالَة** (fi'āla)

**Semantic Profile:**
- **Domains:** Task, Production, Craft
- **Characteristics:** Productive activity, skilled process
- **Vowel Structure:** Kasra + Alif + Ta Marbuta
- **Semantic Features:**
  - Productivity: Very High (1.0)
  - Skill requirement: High (0.8)
  - Process: Very High (1.0)
  - Cognition: Medium (0.5)

**Examples:**
- كِتَابَة (writing) - cognitive-motor task
- زِرَاعَة (agriculture) - productive process
- صِنَاعَة (manufacturing) - craft/industry

---

### 4. Pattern: **فِعَال** (fi'āl)

**Semantic Profile:**
- **Domains:** Social, Conflict, Interaction
- **Characteristics:** Reciprocal, participatory, conflictual
- **Vowel Structure:** Kasra + Alif
- **Semantic Features:**
  - Social interaction: Very High (1.0)
  - Reciprocity: High (0.9)
  - Conflict: High (0.8)
  - Emotional intensity: Medium-High (0.7)

**Examples:**
- قِتَال (fighting) - mutual combat
- صِرَاع (struggle) - competition
- خِصَام (quarrel) - dispute

---

### 5. Pattern: **فِعْل** (fi'l)

**Semantic Profile:**
- **Domains:** Cognitive, Mental
- **Characteristics:** Abstract, intellectual, knowledge-based
- **Vowel Structure:** Kasra + Sukun
- **Semantic Features:**
  - Cognition: Very High (1.0)
  - Abstraction: Very High (1.0)
  - Physicality: Very Low (0.0)
  - Mental processing: Very High (1.0)

**Examples:**
- عِلْم (knowledge) - epistemic state
- فِقْه (understanding/jurisprudence) - deep comprehension
- حِفْظ (memorization) - cognitive retention
- ذِكْر (remembrance) - mental recall

---

### 6. Pattern: **فِعْلَة** (fi'la) / **فَعْلَة** (fa'la)

**Semantic Profile:**
- **Domains:** Single Instance (اسم المرة)
- **Characteristics:** Once, specific occurrence, bounded event
- **Vowel Structure:** Kasra/Fatha + Sukun + Ta Marbuta
- **Semantic Features:**
  - Singularity: Very High (1.0)
  - Boundedness: Very High (1.0)
  - Domain: Inherits from root verb

**Examples:**
- ضَرْبَة (a strike) - single hit
- نَظْرَة (a look) - single glance
- غَضْبَة (fit of anger) - emotional episode
- فِكْرَة (an idea) - mental instance

---

### 7. Pattern: **فُعْلَان** (fu'lān)

**Semantic Profile:**
- **Domains:** Emotional/Physical State
- **Characteristics:** Extended state, fullness, saturation
- **Vowel Structure:** Damma + Sukun + Alif + Nun
- **Semantic Features:**
  - State extension: Very High (1.0)
  - Emotional intensity: High (0.9)
  - Physical manifestation: High (0.8)

**Examples:**
- غُضْبَان (in angry state)
- عَطْشَان (in thirsty state)
- جَوْعَان (in hungry state)

Note: While this is often used as a noun of state (اسم حالة), it represents an important semantic field.

---

### 8. Pattern: **فَعَال** (fa'āl)

**Semantic Profile:**
- **Domains:** Mental/Vocal, Extended Abstract
- **Characteristics:** Sound extension, abstract continuity
- **Vowel Structure:** Fatha + Alif + Long vowel
- **Semantic Features:**
  - Vocal/sound: High (0.8)
  - Abstraction: Medium-High (0.7)
  - Extension: High (0.9)

**Examples:**
- كَلَام (speech) - vocal expression
- خِطَاب (discourse) - extended communication

```sql
    id INTEGER PRIMARY KEY,

    extension DECIMAL(3,2),
    habituality DECIMAL(3,2),
    reciprocity DECIMAL(3,2),
    
    examples TEXT,
    notes TEXT
);
```
